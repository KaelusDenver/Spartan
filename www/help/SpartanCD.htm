Spartan Server Installation

  PLEASE READ THIS FIRST
Your installation package should include:
  • Spartan Installation Media (CD/DVD ROM)
  • National Instruments DIAdem software package

A copy of this document is stored on the Spartan installation media. It may also be easier to print a hard-copy of this document to follow during the various installation steps. There are two primary steps involved to install the Spartan software, and they should be performed in the following order:

  1. Install National Instruments DIAdem using the appropriate installation media included with the installation package.
  2. Install the Spartan software using the appropriate installation media included with the installation package. This document should automatically open when you insert the Spartan installation media. If necessary, you may open the file SpartanCD.htm manually.

This file contains installation directions for installing the Spartan Quality Management System software. Microsoft Windows Server 2003® and Microsoft Windows XP SP4® are the only supported operating systems.

Spartan can be installed as a Hosted or Standalone setup. During installation of Spartan, you will be prompted to select which installation type (Windows Server 2003 will always install as hosted). There are also specific instructions in this procedure for each installation type. Please follow the instructions for your installation type.

The Hosted Setup is the typical installation. The Spartan server program will run as a service on the computer. This enables an "always on" configuration. This installation type is required to install on a Microsoft Windows Server 2003® computer.

The Standalone Setup is used for classified environments or when Spartan will run on only one computer. In this setup, a user must run the Spartan executable each time Spartan is used. This installation is typical for the Microsoft Windows XP SP2® computer.

Please visit the SpartanQMS website for more information or technical support.

Installation Procedure

These steps should be followed in this order by someone logged into the target computer with administrator privileges:

  1. Install DIAdem

    Install the National Instruments DIAdem software using the installation media that came with your copy of Spartan. Proceed with the installation by following the instructions and selecting the default settings when prompted. Please be sure to restart the computer after the DIAdem installation.

  2. Activate DIAdem
    1. Run DIAdem. Use the Windows Start menu to locate the newly installed DIAdem program. Generally it can be found under Start > All Programs > National Instruments > DIAdem {version} > DIAdem.
    2. Activate DIAdem. When DIAdem runs for the first time, it will present a dialog to choose to activate the program. Select the "Activate" button and follow the directions for the activation process.

      DIAdem can be activated over the internet Select "Automatically activate through secure internet connection" then press NEXT.

      When prompted to enter a serial number, please use the number found on the DIAdem license certificate shipped with the installation package.

      You can decide whether to register and receive an email for your records. Hit FINISH when done. Note that DIAdem will restart after activation is complete.
    3. Turn off the DIAdem startup wizard. Check the box "Don't show this screen again", to prevent the DIAdem startup wizard from launching. Click the "Start using DIAdem" to hide the wizard.
    4. Exit DIAdem. Use either the windows close X or the File > Exit menu to exit DIAdem.
  3. Install Spartan
    1. Launch the Spartan installation program. Use a file browser to open and run the file "SpartanCD/setup.exe." (If a security warning appears, Allow and Run rather than saving to disk).

      During the installation, you will be asked to select either a Hosted or Standalone setup. Refer to the description above for more information about which install procedure to use.

      Follow the instructions for the installer. After installation is completed, you must restart your computer before launching Spartan. When the computer has successfully rebooted, login to the computer (again, using an account with administrator privileges).

  4. Authorize Spartan
    1. Access the Spartan Authorization Page. Depending on the setup type it may be necessary to launch the Spartan server first.
      • If Standalone Setup - Run the Spartan application using the "Spartan Server" shortcut (  ) placed on the Desktop. Wait until the application icon (  ) appears in the system tray on the lower right of the desktop.
      • If Hosted Setup - The Spartan service should already be running. No action is required.
    2. Open the Spartan home page. Use the "Spartan Homepage" shortcut placed on the Desktop, or the URL, "http://localhost", can be used as an alternative. If you receive a warning from your browser, the Spartan server may not have started up completely. Please try again in minute.
    3. Login to Spartan. Enter "Administrator" as the username and "Welcome1" for the password then click "Login".
        IMPORTANT
      Please note that the Administrator password should be changed to something secure and easy to remember as soon as convenient. The Administrator password can be changed by clicking the "Profile" link at the top of the Spartan web page, when logged into Spartan as the Administrator.
    4. Contact Summitek for Authorization. Follow the instructions on the authorization page for contacting Summitek Instruments to obtain a Spartan authorization code. Summitek Instruments will require the Site Code found on the authorization page in order to generate a license for your installation.
    5. Enter the Authorization Code. Once the Authorization code is obtained, simply type or copy it into the Authorization Code field and press the "Authorize" button. The authorization page will update to show the new authorization state.

Congratulations Spartan is Now Installed
