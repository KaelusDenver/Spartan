Test Step Modules

Spartan organizes product testing through a series of test modules. These modules define the required instrumentation and physical measurement capabilities of the test. They coordinate measurements from one or more (or fewer!) instruments and control test fixtures for gathering data into the Spartan data repository.

Summitek produces test modules to cover a variety of RF and microwave testing. Test modules are released independently of the Spartan hosted application, so some of the modules you see below may or may not be included on your particular Spartan system.

You can always check SpartanQMS.com for an updated list of available test modules.

Available Modules

  • S Parameter Test Vector or scalar network analysis
  • PIM Test Passive intermodulation distortion analysis
  • DC Attenuation Test Four-point DC resistance
  • Prompt Test Provides instructions and/or pauses
  • Prompted Inputs Test Provides way to collect manual test data
