Users

Spartan requires a login to use. The users panel allows you to add and edit users. The editor requires to enter several fields and assign permissions.

User Parameters

  • Username - The name used to login to Spartan.
  • First Name - The proper name of user.
  • Last Name - The proper name of user.
  • Email - The user's email address.
  • Password - The password used to login to Spartan.
  • Confirm Password - Reenter the password used to login to Spartan.

User Permissions

  • Administrator - This allows unrestricted access to Spartan. Assign this permission carefully.
  • Run Sequences - This allows user to run test sequences.
  • Modify Sequences - This allows user to modify test sequences.
  • Run Scenarios - This allows user to run scenarios.
  • Modify Scenarios - This allows user to modify scenarios.
  • Modify Templates - This allows user to modify templates .
  • Modify Users - This allows user to modify users.
  • Modify Stations - This allows user to modify test stations.
  • Modify Fields - This allows user to modify fields.

User Edit Process

Click on the Configure menu and select Users. The Spartan Users Page is used to show users in Spartan. Click on the green + to add a user to Spartan.

Create a new username based on your Windows login. Be sure to define a password. Assign permissions for the user. Press Save. Repeat as necessary for each user.
