What's new in version 5.0

Spartan 5 is a major new release with a sharpened focus on quality management. We have streamlined the user interface to make the application as intuitive to use as possible. This page describes the new features we've added to make Spartan as useful to you as possible.

Data Mining Scenarios

Spartan 5 introduces data mining scenarios to let you work more easily with the data captured into the system. This includes letting you set up and store complex, multi-level query criteria, and processing the results through a more flexible and extensible template reporting framework.

Editing and running the new data mining scenarios has been made as easy as editing and running test sequences, and much of the user interface functionality has been duplicated to create a consistent experience across the application.

To get started, simply click the Data Mining button on the menu bar.

Filter Specifications

One of the trickiest aspects of S Paremeter test design is creating clear and consistent limit specifications. Over time filter designers have developed a unique and concise terminology to address this complex issue, and we have added it into the S Parameter test module.

More Flexible Templates

Quality management requires a multitude of reporting in various specific formats. Spartan 5 has built upon the idea of reporting templates which can be customized by the user and attached to specific data mining scenarios. The reports output by this system can be used for purposes as varied as manufacturing metrics or test article datasheets.

Templates are added to the system and maintained from the Configure menu. They can be associated with data mining scenarios and attached to specific test sequence steps.
