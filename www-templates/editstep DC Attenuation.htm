This test sequence step uses a Digital Multimeter (DMM) to measure DC attenuation. Resistance is measured between three distinct points:
  • Input to Output (Rab)
  • Input to Ground (Rac)
  • Output to Ground (Rbc)

The test utilizes these measurements to derive electrical attenuation, input and output impedance, and resistor values based on the selected attenuator network topology.

  • Stimulus - Set the DMM stimulus parameters
  • Measure - Select how to calculate the DC Attenuation results
  • Limits - Define pass/fail criteria for the measurements
