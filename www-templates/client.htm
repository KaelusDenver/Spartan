What is the Test Station Client?

Test stations are a unique type of Spartan client computer. Test station clients are used to execute Spartan test sequences. Contrast this with normal Spartan client computers which can be used to edit test sequences, mine the system for data, generate datasheets and analysis reports, etc. The unique property that sets Test Stations apart is the Test Station Client program. The Test Station Client program gets installed to and runs on Spartan test stations and facilitates communications with one or more instruments used during testing.

Uninstalling the Test Station Client

The Spartan client program may be uninstalled from computers which no longer are used to perform Spartan testing. The Spartan Client can be uninstalled from either the Control Panel > Add or Remove Programs, or from the Windows Start menu, Start > All Programs > Kaelus > Spartan Client Uninstall.

Additional Programs and Information

In addition to the Spartan server and client programs, there are several other requirements that test stations must meet in order to be used within the Spartan system.

One basic requirement is that the test station computer must be able to connect to and control any instruments to be used for testing. This requirement is generally met if the computer has already been used for test automation prior to Spartan. The interface bus can be one of several different types: LAN, GPIB, USB, etc. Usually either the instrument manufacturer or the interface bus hardware vendor will supply the necessary software drivers to support computer instrument control.

There are several resources available on the internet which describe test automation, instrument control buses, and types of hardware and software which can be used to control instruments using computers.
